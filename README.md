## Projects

### 1. TTC Infarct Analysis
Python script for automated TTC staining image analysis — quantifies myocardial infarct size by detecting and calculating the ratio of infarcted (white) to viable (red) tissue area from cardiac section images.

**Use case:** Cardiac preservation and ischemia-reperfusion injury research

---

### 2. miRNA Expression Heatmap
Python script for hierarchical clustering and heatmap visualization of miRNA expression data. Performs log2 transformation, euclidean distance-based clustering of both miRNAs and sample groups, and generates a publication-ready clustermap.

**Use case:** miRNA profiling and differential expression analysis in organ preservation and injury research
